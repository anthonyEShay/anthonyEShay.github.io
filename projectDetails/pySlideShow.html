Python Slide Show

About the Python Slide Show Program

This python based program was done to create a more customizable slideshow application than what is provided by default with windows photo viewer. This program can open up a folder and go through all the photos in it either in order or pseudo-randomly. The program also has a listener that keeps track of several user commands like a right arrow (move to next image), space bar (pause), and Q for exiting. The length of time to show each image can be set from any integer value greater or equal to 1.

Notable Features and Technologies

  • GUI Programing with Python Tkinter
  • Uses Python for basic Systems Programing (using windows builtin folder finder)
  • Uses listeners to allow for user interaction
