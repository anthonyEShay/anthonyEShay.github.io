About the Full Stack Django Website

The Full Stack Python Django web project is a full stack website that uses Django to serve dynamic pages, mediate database queries, and control user profiles and actions. This website was done to primarily showcase the powerful capabilities of the Django framework. The completion and showcasing of the website was one part of a final project completed for a graduate level web systems course. This website showcases much of the basic necessary functions a full stack software framework must be able to provide.

This site was designed with function in mind over form. It uses basic Bootstrap to make sure everything is clearly laid out and can be accessed. This site showcases many important standard backend features that are essential for any full stack website. The Django framework was chosen to create the “Shay Family Recipes” website for several reasons. The first is because python is a great programming language that is both powerful and easy to read. Second, Django contains an easy to implement but feature rich user profile system that can be used to accomplish many important user related functions. Third, Django allows for accessing the chosen database in an efficient and succinct manner. Lastly, Django’s view template model makes it possible to dynamically create web pages that utilize database data.

Notable Features and Technologies

  • User sign in, sign up, and logout
  • Storing recipes in a database to allow for dynamic web page generation
  • Allow for searching the database for specific recipe entries
  • Creating a shopping list in local storage and saving it to the database
  • User favorites
  • User comments
