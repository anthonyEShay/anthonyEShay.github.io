    About the Website

    This website is designed to showcase some of the projects I have worked on in my journey to become a skilled programmer. These projects demonstrate my coding abilities in Python, JavaScript, Html/Css, and my ability to apply concepts such as sorting, searching, and multithreading to the programs I write. This online portfolio website is static and built with Html, Css, and JavaScript. The intent was to build a site that has an easy to understand professional layout that allows for a smooth and responsive user experience. The website can be easily navigated with the menu above. The resume page includes an easy to view html resume that can be downloaded as a pdf. The projects page includes several projects, some of which are web based and can be tried from this site.
